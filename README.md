# 統合失調症薬物治療アンケート

GitHub Pages で公開する静的な Web アンケートフォームです。

## 機能

- 同意チェックと基本情報入力
- 統合失調症 問1から問21までの 1-9 評定フォーム
- 各設問で「全て1」または「少なくとも1つ9」を要求する入力チェック
- ブラウザ内の一時保存
- 回答内容の CSV ダウンロードと JSON コピー

## 注意

GitHub Pages は静的ホスティングのため、このフォームだけでは回答をサーバーへ保存しません。回答を中央集約する場合は、Google Forms / Google Sheets Apps Script / Formspree などの送信先を追加してください。
